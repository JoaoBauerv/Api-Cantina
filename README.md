## Instalar
```
c:\xampp\htdocs
git clone https://github.com/JoaoBauerv/Api-Cantina.git
```
## Instalar composer
```
cd api-cantina
composer update
```
## Necessario configurar .env
```
Criar arquivo .env e copiar .envexample, configurar .env, conforme seu banco de dados e servidor local
```

## Iniciar 
```
php artisan serve
```
